Anonymous 21-Jul-20 12:57 PM
yes, but why include that extra m qubits on the y channel? Couldn't we have done a phase kickback only on the n qubits from the x channel?
@Anonymous For phase-kickback to work for only the marked state, we need to use some kind of multi-qubit controlled gate. One example of such a gate is the CNOT gate and TOFFOLI gate (ccx in qiskit). This is because the control part conditions the bit/phase flip for a particular state. Now since we're talking about the Hadamard basis, for a phase filp we need to go from |+> state to |-> state or vice versa. This can either be done using a multi-controlled Z gate on |0> state or everything can be brought into Hadamard basis and then multi-controlled X gate can be used for the phase-kickback.
