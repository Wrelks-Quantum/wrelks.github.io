@Anonymous giving the "proof" of no-cloning theorem would be like telling you how to swim by demonstrating on sand but i can try to give an intuitive idea to see where it all stems from. so, to get an idea there are a few things you need to keep in mind. 1. arbitrary unknown state -> you don't know anything about it 2. there are two flavours? of this theorem. we usually prove it for pure states -> no cloning theorem. we can also do it for mixed states -> no-broadcast theorem. it's a corollary of the no cloning theorem. (there's also the fancy no-deletion. it's like going down the rabbit hole 3. this is how the proof goes step 01 : we have two systems. one is for Alice, the other for Bob. and alice wants to copy an unknown arbitrary state (she has no prior knowledge about it). She wants to try and find a way with which she can copy this state "IDK_alice" into Bob's system. For that she starts by taking a blank state from Bob's system "BLANK_bob". and she's aiming for the product state |IDK_alice BLANK_bob> step 02 : Alice is given a critical choice between two operations she can do to manipulate her IDK_alice state. option A: she can observe it (if she does it, all is doomed. the state will collapse and she doesnt want it to happen) option B : she could be patient and wait for |IDK_alice BLANK_bob> combined system to evolve in time by controlling something called the Hamiltonian of the combined system. she's a wise girl, so chooses B(edited)

I guess you have to go back to the idea that grover's algorithm performs a search over an unstructured database with the eventual goal of providing the position (or maybe better referred to as the index) of the element you're interested in within that unstructured database. So there are actually two nuggets of information here: the value of your desired element and its index. You know the value but you don't know its index (or if it actually exists) within the database. Now the crucial part is that when you construct the oracle, you only make use of the former (value). For example, for the phase oracle in the qiskit textbook), we used control z gates to mark elements with the values that we are interested in. When we then run the algorithm, we then get the index.(edited)
