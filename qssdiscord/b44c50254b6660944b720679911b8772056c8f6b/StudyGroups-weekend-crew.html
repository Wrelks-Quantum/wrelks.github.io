Anonymous 25-Jul-20 12:23 PM
I think it comes out of the Oracle Uf0 used in the diffuser circuit. So the Uf0 Oracle is just finding the marked state in the 0 bit string and flipping it. In diffuser circuit, we sandwich Uf0 between two hadamard gates and the resultant bit string expression comes from the Uf0 but the effect of added hadamard gates is reflection about the average amplitude of the whole bit string.
